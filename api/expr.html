template<size_t N>
inline args_type<N> args()

Given a template parameter N, returns a tuple of size N that contains ArgExpr for each element. This function can be used in combination with std::tie or structured binding to quickly assign names to argument expression. For example, to bind four arguments to the names n, A, B, and C:

auto [n, A, B, C] = kernel_launcher::args<4>();

The above example only works in C++17 or higher. For, C++11 or higher, one can use std::tie as follows:

ArgExpr n, A, B, C;
std::tie(n, A, B, C) = kernel_launcher::args<4>();

template<typename C, typename... Es> KERNEL_LAUNCHER_DEVICE_ATTRIBUTES_FORALL(KERNEL_LAUNCHER_DEFINE_DEVICE_ATTRIBUTE) struct SelectExpr SelectExpr select (C &&index, Es &&... operands)

Returns a new expression that first evaluates index as an integer n and then evaluates the n-th operand in the list of operand expressions. For example, if index returns 0, then the first operand is evaluated. Note that index can also be a boolean expression, in which case false is interpreted as 0 and true is interpreted as 1.

Parameters:

  • index – The index expression

  • operands – The operand expressions.

template<typename C, typename Es>
SelectExpr index(C &&cond, Es &&operands)

Returns a new expression that first evaluates index as an integer n and then evaluates the n-th operand in the list of operand expressions. For example, if index returns 0, then the first operand is evaluated.

This function is equivalent to select, expect it takes an iterable (e..g, vector, array) instead of a variadic list of arguments.

Parameters:

  • index – The index expression

  • operands – The operand expressions.

Returns a new expression that first evaluates cond and then evaluates either true_expr if the condition is true and false_expr otherwise.

Parameters:

  • true_expr – The expression if the condition is true.

  • false_expr – The expression if the condition is false.
